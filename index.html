Contribute

Sources on GitHub
It is very very easy to contribute to this webiste. Have already contributed: (in chronological order)
Valerio Nappi (Author) - Giacomo Tombolan - Simone Guglielmino - Andrea Bertazzoni - Francesco Lenzi - Donato Carlo Giorgio - More to come...
If you have some material of which you own rights and you're willing to mke it available to everyone, or if you know some resources that might be useful to be added in this page, feel free to open a Pull request. If you don't know the tools, get in contact with me opening an issue and it will be a pleasure to help.
